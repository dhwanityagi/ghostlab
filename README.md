# GhostLab — Digital Twin Energy Simulation System

## 🚀 Overview
GhostLab is a backend-driven digital twin system designed to simulate hostel room conditions and predict electricity usage, cost, and comfort levels in real time.

The system models environmental inputs and generates actionable insights through predictive analytics.

---

## 🧠 Problem
Energy usage in residential spaces is often inefficient due to lack of visibility into consumption patterns and environmental impact.

---

## 💡 Solution
GhostLab simulates room conditions and predicts electricity usage using structured data and modeling techniques, enabling better decision-making.

---

## ⚙️ System Design
- Data ingestion for temperature, humidity, usage signals
- Prediction layer for electricity cost and comfort
- Scenario simulation engine
- API-based backend (FastAPI)

---

## 📊 Key Results
- Processed 10,000+ telemetry data points  
- Achieved ~72–88% prediction accuracy  
- Simulated ~18% potential energy waste reduction  

---

## 🛠 Tech Stack
FastAPI, SQLite, Predictive Modeling
